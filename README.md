# 🚀 FED — Federation of Entrepreneurship Development

A modern, responsive website for FED (Federation of Entrepreneurship Development), KIIT University — built with pure HTML, CSS, and JavaScript.

---

## 🌐 Live Preview

> Open `index.html` directly in any browser — no server required.

---

## ✨ Features

- **Dark / Light Mode** toggle with smooth transitions
- **Fully Responsive** — works on mobile, tablet, and desktop
- **Scroll Reveal Animations** for all sections
- **Sticky Navbar** with hamburger menu on mobile
- **Interactive Contact Form** with submit feedback
- **No external dependencies** — pure HTML, CSS, JS
